# Quick Start

## 🌱 快速开始

#### 5 分钟拥有你的链上农场

只要 3 步，新手小白也能立刻开始种田赚钱！

#### 1. 连接钱包（30 秒）

[立即连接钱包](https://yourgame.com)

支持钱包：\
• MetaMask • Trust Wallet • OKX Wallet • Binance Wallet • WalletConnect

> 第一次玩？手机用户推荐直接用 Trust Wallet 或 OKX Wallet，体验最好。

#### 2. 领取新手礼包（免费）

连接成功后自动弹出：\
**新玩家专属礼包**

* 3 块稀有土地 NFT（永久归你）
* 1000 颗新手种子（向日葵、西红柿、玉米）
* 7 天产量 ×3 倍增卡
* 500 $CROP 启动资金

::callout{title="💡 小贴士" color="#FFB300"} 如果没自动弹出，直接点页面右上角 “Claim Starter Pack” 按钮即可领取 ::

#### 3. 开始种田（超简单）

1. 选择空地 → 点击「耕地」
2. 选择种子 → 点击「播种」
3. 等待成熟（8-24 小时）→ 点击「收获」→ $CROP 自动到账
4. 收获的 $CROP 可直接提现到钱包，也可继续购买更多种子/动物
